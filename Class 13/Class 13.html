To finish off let’s make a nicer volcano plot. - Add the log2 threshold lines at +2/-2 - Add P-value threshold lines at 0.05 - Add color to highlight the subset of genes that meet both of the above thresholds

mycols <- rep("gray", nrow(res))
mycols[res$log2FoldChange >=2] <- "lightblue"
mycols[res$log2FoldChange <= -2] <- "lightblue"
mycols[res$padj > 0.05] <- "grey"
ggplot(res)+
